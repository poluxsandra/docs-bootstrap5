Opções

Personalize rapidamente o Bootstrap com variáveis integradas para alternar facilmente as preferências CSS globais para controlar o estilo e o comportamento.

Personalize o Bootstrap com nosso arquivo de variáveis ​​personalizadas integrado e alterne facilmente as preferências globais de CSS com as novas variáveis Sass $enable-*. Substitua o valor de uma variável e recompile com o comando npm run test conforme necessário.

Você pode localizar e personalizar essas variáveis ​​para as principais opções globais no arquivo scss/_variables.scss do Bootstrap.

Variáveis Valores Descrição
$spacer 1rem (padrão), ou qualquer valor > 0 Especifica o valor padrão do espaçador para gerar programaticamente nossos utilitários de espaçamento.
$enable-rounded true (padrão) ou false Ativa os estilos predefinidos por border-radius em vários componentes.
$enable-shadows true ou false (padrão) Ativa os estilos decorativos predefinidos por box-shadow em vários componentes. Não afeta os box-shadow's usados em estados de foco.
$enable-gradients true ou false (padrão Ativa os estilos de gradiente predefinidos por meio do background-image em vários componentes.
$enable-transitions true (padrão) ou false Ativa as predefinições por meio dos transition's em vários componentes.
$enable-reduced-motion true (padrão) ou false Ativa a consulta de mídia de prefers-reduced-motion, que suprime certas animações/transições com base nas preferências do navegador/sistema operacional dos usuários.
$enable-grid-classes true (padrão) ou false Permite a geração de classes CSS para o sistema de grade (por exemplo .row, .col-md-1, etc.).
$enable-caret true (padrão) ou false Ativa o acento circunflexo do pseudoelemento .dropdown-toggle.
$enable-button-pointers true (padrão) ou false Adicione o cursor de “mão” a elementos de botão não desabilitados.
$enable-rfs true (padrão) ou false Ativa globalmente o RFS.
$enable-validation-icons true (padrão) ou false Ativa ícones por meio de background-image em entradas textuais e alguns formulários personalizados para estados de validação.
$enable-negative-margins true ou false (padrão) Ativa a geração do utilitário de margem negativa.
$enable-deprecation-messages true (padrão) ou false Defina como false para ocultar avisos ao usar qualquer um dos mixins e funções obsoletos que estão planejados para serem removidos na v6.
$enable-important-utilities true (padrão) ou false Ativa o sufixo !important nas classes de utilitários.
