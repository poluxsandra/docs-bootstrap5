Variáveis ​​CSS

Use as propriedades personalizadas CSS do Bootstrap para um design e desenvolvimento rápido e voltado para o futuro.

Bootstrap inclui cerca de duas dúzias de propriedades customizadas CSS (variáveis) em seu CSS compilado, com dezenas mais a caminho para customização aprimorada por componente. Eles fornecem acesso fácil aos valores comumente usados, como nossas cores de tema, pontos de interrupção e pilhas de fontes primárias ao trabalhar no inspetor de seu navegador, uma caixa de proteção de código ou prototipagem geral.

Todas as nossas propriedades personalizadas são prefixadas com bs- para evitar conflitos com CSS de terceiros.

Variáveis ​​de Componente

Também estamos começando a usar propriedades personalizadas como variáveis ​​locais para vários componentes. Dessa forma, podemos reduzir nosso CSS compilado, garantir que os estilos não sejam herdados em lugares como tabelas aninhadas e permitir um pouco de restyling e extensão de componentes do Bootstrap após a compilação do Sass.

Dê uma olhada em nossa documentação de tabela para alguns insights sobre como estamos usando variáveis ​​CSS..

Também estamos usando variáveis ​​CSS em nossas grades - principalmente para calhas - com mais uso de componentes chegando no futuro.

Exemplos

As variáveis ​​CSS oferecem flexibilidade semelhante às variáveis ​​do Sass, mas sem a necessidade de compilação antes de serem veiculadas no navegador. Por exemplo, aqui estamos redefinindo os estilos de fonte e link de nossa página com variáveis ​​CSS. 

body {
    font: 1rem/1.5 var(--bs-font-sans-serif);
  }
  a {
    color: var(--bs-blue);
  }
