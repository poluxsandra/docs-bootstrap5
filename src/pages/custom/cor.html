Notas sobre Sass

O Sass não pode gerar variáveis ​​de maneira programática, portanto, nós mesmos criamos variáveis ​​manualmente para cada tom e tonalidade. Especificamos o valor do ponto médio (por exemplo $blue-500) e usamos funções de cores personalizadas para tingir (clarear) ou sombrear (escurecer) nossas cores por meio da função de cor mix(), nativa do Sass.

Usar a função mix() não é a mesma coisa que usar as funções lighten() e darken() — a primeira combina a cor especificada com branco ou preto, enquanto as últimas apenas ajustam o valor de claridade de cada cor. O resultado é um conjunto de cores muito mais completo, como mostrado nesta demonstração do CodePen.

Nossas funções tint-color() e shade-color() usam a função mix() ao lado de nossa variável $theme-color-interval, que especifica um valor percentual escalonado para cada cor mista que produzimos. Consulte os arquivos scss/_functions.scss e scss/_variables.scss para obter o código fonte completo.

Mapas de Cor do Sass

Os arquivos-fonte Sass do Bootstrap incluem três mapas para ajudá-lo a percorrer de forma rápida e fácil uma lista de cores e seus valores hexadecimais.

  • $colors lista todas as cores base (500) disponíveis
  • $theme-colors lista todas as cores do tema semanticamente nomeadas (mostrado abaixo)
  • $grays lista todas as matizes e tons de cinza

Dentro do arquivo scss/_variables.scss, você encontrará as variáveis ​​de cores do Bootstrap e o mapa Sass $colors: 

$colors: (
        "blue":       $blue,
        "indigo":     $indigo,
        "purple":     $purple,
        "pink":       $pink,
        "red":        $red,
        "orange":     $orange,
        "yellow":     $yellow,
        "green":      $green,
        "teal":       $teal,
        "cyan":       $cyan,
        "white":      $white,
        "gray":       $gray-600,
        "gray-dark":  $gray-800
      );

Adicione, remova ou modifique valores dentro do mapa para atualizar como eles são usados ​​em muitos outros componentes. Infelizmente, no momento, nem todos os componentes utilizam este mapa Sass. As atualizações futuras se esforçarão para melhorar isso. Até então, planeje fazer uso das variavéis ${color} e este mapa Sass.

Exemplo

Veja como você pode usar isso em seu Sass:

.alpha { color: $purple; }
      .beta {
        color: $yellow-300;
        background-color: $indigo-900;
      }

Classes de utilitários de Cores e Fundo também estão disponíveis para configurar color e background-color usando os valores das cores 500.
