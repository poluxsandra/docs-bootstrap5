Sistema de Grade(Grid)

Use nossa poderoso sistema de grade Flexbox com a política para construir layouts de todas as formas e tamanhos graças a um sistema de doze colunas, seis níveis responsivos padrões, variáveis ​​Sass e Mixins, e dezenas de classes predefinidas.

Exemplo

O sistema de grade do Bootstrap usa uma série de contêiners, linhas e colunas para fazer o layout e alinhar o conteúdo. Ele é construído com Flexbox Abaixo está um exemplo e uma explicação detalhada de como o sistema de grade funciona.

Como funciona

Resumindo, é assim que o sistema de grade se reúne:

  • Nossa grade suporta seis breakpoints responsivos.
    Breakpoints são baseados em consultas de mídia de min-width, o que significa que afetam aquele breakpoint e todos aqueles acima dele (por exemplo, .col-sm-4 aplica-se à sm, md, lg, xl, e xxl). Isso significa que você pode controlar o tamanho e o comportamento de contêiners e colunas de cada breakpoint.

  • Os contêiners centralizam e alinham horizontalmente seu conteúdo.
    Use .container para uma largura de pixel responsiva, .container-fluid para width: 100% através de todos viewports e dispositivos, ou um contêiner responsivo (por exemplo, .container-md) para uma combinação de larguras de pixel responsiva e fluida.

  • As linhas (Rows) são invólucros para colunas.
    Cada coluna tem uma horizontal padding (chamada de gutter) para controlar o espaço entre elas. Esse padding é então neutralizado nas linhas com margens negativas para garantir que o conteúdo em suas colunas esteja visualmente alinhado no lado esquerdo. As linhas também oferecem suporte a classes modificadoras para aplicar uniformemente o dimensionamento de coluna e classes de gutters para alterar o espaçamento de seu conteúdo.

  • As colunas são incrivelmente flexíveis.
    Existem 12 colunas de modelo disponíveis por linha, permitindo que você crie diferentes combinações de elementos que abrangem qualquer número de colunas. As classes de coluna indicam o número de colunas do modelo a serem estendidas (por exemplo, col-4 abrange 4). Larguras são definidas em porcentagens para que você sempre tenha o mesmo tamanho relativo.

  • Os Gutters também são responsivas e personalizáveis.
    As classes de gutters estão disponíveis em todos breakpoints, com todos eles com o mesmo tamanho de margem e espaçamento do padding. Pode-se alterar gutters horizontais com classes .gx-*, e gutters verticais com classes .gy-*, ou todos os gutters com classes .g-*. As classes .g-0 também estão disponíveis para remover gutters.

  • Variáveis, mapas e mixins de Sass aumentam o poder do sistema de grade.
    Se você não quiser usar as classes de grade predefinidas no Bootstrap, você pode usar a fonte de Sass do sistema de grade para criar a sua própria marcação semântica. Também incluímos algumas propriedades customizadas CSS para consumir essas variáveis ​​Sass para uma flexibilidade ainda maior para você.

Esteja ciente das limitações e bugs em torno do flexbox, como a incapacidade de usar alguns elementos HTML como contêiners flex.

Opções de Grade

O sistema de grade do Bootstrap pode se adaptar a todos os seis breakpoints padrões, e a quaisquers breakpoints personalizados por você. As seis camadas de grade padrão são as seguintes:

  • Muito Pequeno (xs)
  • Pequeno (sm)
  • Médio (md)
  • Grande (lg)
  • Muito Grande (xl)
  • Gigante (xxl)

Conforme observado acima, cada um desses breakpoints tem seus próprios contêiner, prefixo de classe e modificadores. Veja como a grade muda entre esses breakpoints:

Sass

Ao usar os arquivos-fonte Sass do Bootstrap, você tem a opção de usar variáveis ​​e mixins do Sass para criar layouts de página personalizados, semânticos e responsivos. Nossas classes de grade predefinidas usam essas mesmas variáveis ​​e mixins para fornecer um conjunto completo de classes prontas para uso para layouts de resposta rápida.

Variáveis

Variáveis ​​e mapas determinam o número de colunas, a largura do gutter e o ponto de consulta de mídia no qual as colunas flutuantes devem começar. Nós os usamos para gerar as classes de grade predefinidas documentadas acima, bem como para os mixins personalizados listados abaixo. 

$grid-columns:      12;
  $grid-gutter-width: 1.5rem;
  
$grid-breakpoints: (
    xs: 0,
    sm: 576px,
    md: 768px,
    lg: 992px,
    xl: 1200px,
    xxl: 1400px
  );
  
$container-max-widths: (
    sm: 540px,
    md: 720px,
    lg: 960px,
    xl: 1140px,
    xxl: 1320px
  );

Mixins

Mixins são usados ​​em conjunto com as variáveis ​​de grade para gerar CSS semântico para colunas de grade individuais. 

  // Cria um wrapper para uma série de colunas
  @include make-row();
  
  // Deixe o elemento pronto para a grade (aplicando tudo, menos a largura)
  @include make-col-ready();
  
  // Sem valores de tamanho opcionais, o mixin criará colunas iguais (semelhante com o uso de .col)
  @include make-col();
  @include make-col($size, $columns: $grid-columns);
  
  // Divirta-se compensando ou alterando a ordem de classificação
  @include make-col-offset($size, $columns: $grid-columns);

Exemplo de Uso

Você pode modificar as variáveis ​​para seus próprios valores personalizados, ou apenas usar os mixins com seus valores padrão. Aqui está um exemplo de como usar as configurações padrão para criar um layout de duas colunas com uma lacuna entre elas. 

.exemplo-contêiner {
    @include make-container();
    // Certifique-se de definir esta largura após o mixin substituto
    // "largura: 100%` gerado por 'make-container()'"
    width: 800px;
  }
  
  .exemplo-linha(row) {
    @include make-row();
  }
  
  .exemplo-conteúdo-principal(main) {
    @include make-col-ready();
  
    @include media-breakpoint-up(sm) {
      @include make-col(6);
    }
    @include media-breakpoint-up(lg) {
      @include make-col(8);
    }
  }
  
  .examplo-conteúdo-secundário {
    @include make-col-ready();
  
    @include media-breakpoint-up(sm) {
      @include make-col(6);
    }
    @include media-breakpoint-up(lg) {
      @include make-col(4);
    }
  }

Personalizando a Grade

Usando nossas variáveis ​​e mapas Sass de grade integrados, é possível personalizar completamente as classes de grade predefinidas. Altere o número de camadas, as dimensões da consulta de mídia e as larguras dos contêiners - depois recompile.

Colunas e gutters

O número de colunas da grade pode ser modificado por meio de variáveis ​​Sass. $grid-columns é usado para gerar as larguras (em porcentagem) de cada coluna individual enquanto $grid-gutter-width define a largura dos gutters das colunas. 

$grid-columns: 12 !default;
  $grid-gutter-width: 1.5rem !default;

Camadas de Grade

Indo além das próprias colunas, você também pode personalizar o número de camadas da grade. Se você quisesse apenas quatro camadas de grade, atualizaria o $grid-breakpoints e o $container-max-widths para algo assim: 

$grid-breakpoints: (
    xs: 0,
    sm: 480px,
    md: 768px,
    lg: 1024px
  );
  
  $container-max-widths: (
    sm: 420px,
    md: 720px,
    lg: 960px
  );

Ao fazer qualquer alteração nas variáveis ​​ou mapas do Sass, você precisará salvar suas alterações e recompilar. Isso produzirá um novo conjunto de classes de grade predefinidas para larguras de coluna, deslocamentos e ordenação. Os utilitários de visibilidade responsiva também serão atualizados para usar os pontos de interrupção personalizados. Certifique-se de usar valores de grade em px (não em rem, em, ou %).
