Utilitários Para Layout

Para um desenvolvimento mais responsivo e compatível com dispositivos móveis, o Bootstrap inclui dezenas de classes de utilitários para mostrar, ocultar, alinhar e espaçar o conteúdo.

Mudando o display

Use nossos utilitários de exibição para alternar responsavelmente os valores comuns da propriedade display. Misture-os com nosso sistema de grade, conteúdos ou componentes para mostrá-los ou ocultá-los em janelas de exibição específicas.

Opções de Flexbox

O Bootstrap é construído com o flexbox, mas nem todas propriedades display de elementos foram mudadas para display:flex pois isso adicionaria muitas substituições desnecessárias e mudaria inesperadamente os principais comportamentos do navegador. A maioria de nossos componentes são construídos com flexbox habilitado.

Se você precisar adicionar o valor display:flex em um elemento, faça-o usando a classe .d-flex ou uma das variantes responsivas (por exemplo, .d-sm-flex). Você precisará dessa classe ou do valor da propriedade display para permitir o uso de nossos utilitários extras do flexbox para dimensionamento, alinhamento, espaçamento e muito mais.

Margin e Padding

Use os margin, padding e utilitários de espaçamento para controlar como os elementos e componentes são espaçados e dimensionados. O Bootstrap inclui uma escala de seis níveis para utilitários de espaçamento, com base em um valor padrão de 1rem para a variável $spacer. Escolha valores para todos os viewports (por exemplo, .me-3 para margin-right: 1rem em LTR), ou escolha variantes responsivas para visar viewports específicos (por exemplo, .me-md-3 para margin-right: 1rem — em LTR — começando no breakpoint md).

Alternância de visibility

Quando a alternância do valor da propriedade display não é necessária, você pode alternar o valor da propriedade visibility de um elemento com nossos utilitários de visibilidade. Os elementos invisíveis ainda afetarão o layout da página, mas estarão visualmente ocultos para os visitantes.
