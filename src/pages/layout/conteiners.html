Contêiners

Os contêineres são blocos de construção fundamentais do Bootstrap que contém, preenchem e alinham seu conteúdo em um determinado dispositivo ou viewport.

Como Eles Trabalham

Os contêiners são os elementos de layout mais básicos no Bootstrap e são necessários ao usar nosso sistema de grade padrão. Os contêiners são usados ​​para conter, preencher e (às vezes) centralizar o conteúdo dentro deles. Embora os contêiners possam ser aninhados, a maioria dos layouts não exige um contêiner aninhado.

Bootstrap vem com três contêineres diferentes:

  • .container, que define um max-width em breakpoint responsivo
  • .container-fluid, que define width: 100% em todos os breakpoints
  • .container-{breakpoint}, que define width: 100% até o breakpoint especificado

Contêiner Padrão

Nossa classe padrão .container é um contêiner responsivo de largura fixa, o que significa que sua max-width muda em cada breakpoint. 

<div class="container">
        <!-- Conteúdo aqui -->
      </div>

Contêiners Responsivos

Os contêineres responsivos permitem que você especifique uma classe com 100% de largura até que o breakpoint especificado seja alcançado, após o qual aplicamos as max-width's para cada um dos breakpoints maiores. Por exemplo, .container-sm possui 100% de largura para começar até que o breakpoint sm seja alcançado, onde aumentará com md, lg, xl, e xxl. 

      <div class="container-sm">100% de largura até o breakpoint pequeno(small)</div>
      <div class="container-md">100% de largura até o breakpoint médio(medium)</div>
      <div class="container-lg">100% de largura até o breakpoint grande(large)</div>
      <div class="container-xl">100% de largura até o breakpoint muito grande (x-large)</div>
      <div class="container-xxl">100% de largura até o breakpoint gigante(xx-large)</div>

Contêiners Fluidos

Use .container-fluid para um contêiner de largura total, abrangendo todo o viewport.

<div class="container-fluid">
        ...
      </div>

Sass

Conforme mostrado acima, o Bootstrap gera uma série de classes de contêiners predefinidas para ajudá-lo a construir os layouts que você deseja. Você pode personalizar essas classes de contêiners predefinidas, modificando o mapa Sass (encontrado em _variables.scss) que as fornece: 

$container-max-widths: (
        sm: 540px,
        md: 720px,
        lg: 960px,
        xl: 1140px,
        xxl: 1320px
      );

Além de personalizar o Sass, você também pode criar seus próprios contêiners com nosso mixin Sass.

// Código-Fonte do Mixin
      @mixin make-container($padding-x: $container-padding-x) {
        width: 100%;
        padding-right: $padding-x;
        padding-left: $padding-x;
        margin-right: auto;
        margin-left: auto;
      }
      
      // Uso
      .custom-container {
        @include make-container();
      }

Para obter mais informações e exemplos sobre como modificar nossos mapas e variáveis ​​Sass, consulte a seção Sass da documentação do Grid..
