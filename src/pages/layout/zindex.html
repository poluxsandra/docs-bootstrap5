Z-index

Embora não façam parte do sistema de grade do Bootstrap, o Z-index desempenham um papel importante em como nossos componentes se sobrepõem e interagem uns com os outros.

Vários componentes do Bootstrap utilizam a propriedade CSS Z-index, que ajuda a controlar o layout, fornecendo um terceiro eixo para organizar o conteúdo. Usamos uma escala padrão de Z-index no Bootstrap que foi projetada para a navegação em camadas, tooltips, popovers, modais e muito mais.

Esses valores mais altos começam em um número arbitrário, alto e específico o suficiente para evitar conflitos. Precisamos de um conjunto padrão desses em nossos componentes em camadas - dicas de ferramentas, popovers, barras de navegação, menus suspensos, modais - para que possamos ser razoavelmente consistentes nos comportamentos. Não há razão para não termos usado 100+ ou 500+.

Não encorajamos a personalização desses valores individuais; se você mudar um, provavelmente precisará mudar todos. 

$zindex-dropdown:                   1000;
      $zindex-sticky:                     1020;
      $zindex-fixed:                      1030;
      $zindex-modal-backdrop:             1040;
      $zindex-offcanvas:                  1050;
      $zindex-modal:                      1060;
      $zindex-popover:                    1070;
      $zindex-tooltip:                    1080;

Para lidar com bordas sobrepostas dentro de componentes (por exemplo, botões e entradas em grupo de botões), usamos um valor de z-index de único dígito (como 1, 2, e 3 por padrão, em estados hover ou ativos). No hover/foco/ativo, colocamos um elemento específico em primeiro plano com um valor de z-index alto para mostrar sua borda sobre os elementos irmãos.
