RFS

O mecanismo de redimensionamento do Bootstrap dimensiona responsavelmente propriedades CSS comuns para melhor utilizar o espaço disponível em viewports e dispositivos.

O que é RFS?

O projeto paralelo do Bootstrap, RFS é um mecanismo de redimensionamento de unidade que foi elaborado para redimensionar tamanhos de fonte (daí sua abreviatura para Tamanhos de Fonte Responsivos). Atualmente o RFS é capaz de ser especificada na maioria das propriedades CSS com valores unitários como margin, padding, border-radius, ou mesmo box-shadow.

O mecanismo automaticamente calcula os valores fornecidos com base nas dimensões do viewport do navegador. Ele será compilado através da combinação das funções calc() com a unidade rem, e unidades de viewport para permitir o comportamento de dimensionamento responsivo.

Usando RFS

Os mixins estão incluídos no Bootstrap e estarão disponíveis assim que você incluir o scss do Bootstrap. O RFS também pode ser instalado separadamente caso necessário.

Usando os mixins

O mixin rfs() possui atalhos para font-size, margin, margin-top, margin-right, margin-bottom, margin-left, padding, padding-top, padding-right, padding-bottom, and padding-left. Veja o exemplo abaixo para o código-fonte Sass e CSS compilado. 

.title {
  @include font-size(4rem);
}

.title {
  font-size: calc(1.525rem + 3.3vw);
}

@media (min-width: 1200px) {
  .title {
    font-size: 4rem;
  }
}

Qualquer outra propriedade pode ser passada para o mixin rfs() desta forma: 

.selector {
  @include rfs(4rem, border-radius);
}

!important também pode ser adicionado a qualquer valor que você quiser: 

.selector {
  @include padding(2.5rem !important);
}

Usando as funções

Quando você não quiser usar as inclusões, também existem duas outras funções:

  • rfs-value() converte para um valor em rem se um valor em px é passado, em outros casos retorna o mesmo resultado.
  • rfs-fluid-value() retorna uma versão fluida de um valor se a propriedade precisar ser redimensionada.

Neste exemplo, usamos um dos mixins embutidos do Bootstrap (breakpoint responsivo) para aplicar somente o estilo do breakpoint lg abaixo. 

.selector {
  @include media-breakpoint-down(lg) {
    padding: rfs-fluid-value(2rem);
    font-size: rfs-fluid-value(1.125rem);
  }
}
@media (max-width: 991.98px) {
  .selector {
    padding: calc(1.325rem + 0.9vw);
    font-size: 1.125rem; /* 1.125rem é muito pequeno, então o RFS não redimensionará*/
  }
}
