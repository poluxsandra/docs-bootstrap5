Esta é a forma mais básica de Bootstrap: arquivos pré-compilados para uso rápido em quase qualquer projeto da web. Nós fornecemos CSS e JS compilados (bootstrap.*), bem como CSS e JS compilados e minimificados (bootstrap.min.*). Mapas de Fontes (bootstrap.*.map) estão disponíveis para uso com ferramentas de desenvolvedor dos navegadores. Os arquivos JS agrupados (bootstrap.bundle.js) (e minificados bootstrap.bundle.min.js) incluem o Popper.

Bootstrap Código-Fonte

O download do código-fonte do Bootstrap inclui os recursos CSS e JavaScript pré-compilados, junto com o Sass, JavaScript e documentos de fonte. Mais especificamente, inclui o seguinte e mais: 

bootstrap/
├── dist/
│   ├── css/
│   └── js/
├── site/
│   └──content/
│      └── docs/
│          └── 5.0/
│              └── examples/
├── js/
└── scss/

O scss/ e o js/ são os códigos-fonte de nosso CSS e JavaScript.
A pasta dist/ inclui tudo o que está listado na seção de download pré-compilado acima.
A pasta site/docs/ inclui o código-fonte da nossa documentação;
E a pasta examples/ mostra exemplos de como usar o Bootstrap.
Além disso, qualquer outro arquivo incluído oferece suporte para pacotes, informações de licença e desenvolvimento.
