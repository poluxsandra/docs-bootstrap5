CSS e JS Compilado

Baixe o código compilado pronto para usar do Bootstrap v5.0.0-beta2 para adicionar facilmente em seu projeto, que inclui:

  • Compilado e minificado arquivos CSS (veja uma comparação de arquivos CSS)
  • Compilado e minificado JavaScript plugins (veja uma comparação de arquivos JS)

Isso não inclui documentos, arquivos de origem ou quaisquer dependências opcionais de JavaScript, como Popper.

Arquivos Fonte

Compile o Bootstrap com seu próprio pipeline de ativos baixando nossos arquivos de origem Sass, JavaScript e de documentos. Esta opção requer algumas ferramentas adicionais:

  • Compilador SASS para compilar arquivos fonte Sass em arquivos CSS
  • Autoprefixador para prefixação de fornecedores CSS

Se você precisar de nosso conjunto completo de Ferramentas de Construção, elas estão incluídas para desenvolver o Bootstrap e seus documentos, mas provavelmente são inadequados para seus próprios fins.

Gerenciadores de pacotes

Importe os arquivos fonte do Bootstrap para quase todos os projetos com alguns dos gerenciadores de pacotes mais populares. Não importa o gerenciador de pacotes, o Bootstrap exigirá um Compilador SASS e um Autoprefixador para uma configuração que corresponde às nossas versões compiladas oficiais.

npm

Instale o Bootstrap em seu Node.js com o software npm: 

npm install bootstrap@next

const bootstrap = require('bootstrap') ou import bootstrap from 'bootstrap' carregará todos os plug-ins do Bootstrap em um bootstrap objeto. O próprio módulo bootstrap exporta todos os nossos plug-ins. Você pode carregar manualmente os plug-ins do Bootstrap individualmente carregando os arquivos /js/dist/*.js no diretório de nível superior do pacote.

O package.json do Bootstrap contém alguns metadados adicionais nas seguintes chaves:

  • sass - caminho para o arquivo-fonte SASS principal Bootstrap
  • style - caminho para o CSS não minimizado do Bootstrap que foi pré-compilado usando configurações padrão (sem personalização)
Comece com Bootstrap via npm com nosso projeto inicial!

Vá para twbs/bootstrap-npm-starter para ver como construir e personalizar o Bootstrap em seu próprio projeto npm. Inclui compilador Sass, Autoprefixador, Stylelint, PurgeCSS e ícones de bootstrap.

yarn

Instale o Bootstrap em seu Node.Js através do pacote yarn: 

yarn add bootstrap@next

RubyGems

Instale o Bootstrap no seu Ruby usando o Bundler (Recomendado) e RubyGems adicionando a seguinte linha ao seu Gemfile: 

gem 'bootstrap', '~> 5.0.0.beta2'

Como alternativa, se você não estiver usando o Bundler, você pode instalar pelo gem através deste comando: 

gem install bootstrap -v 5.0.0.beta2

Veja o README do Gem para mais detalhes.

Composer

Você também pode instalar e gerenciar o Sass e o JavaScript do Bootstrap usando o Composer: 

composer require twbs/bootstrap:5.0.0-beta2

NuGet

Se você é desenvolvedor .NET, você pode instalar e gerenciar os arquivos do Bootstrap: CSS ou Sass e JavaScript, usando o NuGet: 

Install-Package bootstrap
Install-Package bootstrap.sass
